# Whimsical 1800s Game Assets

All raster artwork in this directory is original project artwork created with the built-in ImageGen workflow. No commercial Ticket to Ride artwork is included or reused.

The shared art direction was a whimsical late-1800s railway adventure: hand-painted storybook realism, engraved brass and leather details, warm period color, soft atmospheric light, no text, and no logos.

## Surfaces

- `board-map.webp`: large aerial tabletop map of the continental United States and southern Canada, richly textured terrain and waterways, clear central landmass, open composition for interactive route overlays.
- `table-background.webp`: dark teal leather and carved walnut railway table with subtle brass fittings and restrained edge detail.
- `ticket-face.webp`: horizontal cream railway destination ticket with engraved border, faded route-map filigree, wax-red accents, and a clear center for live route text.
- `ticket-back.webp`: stacked horizontal destination-ticket back with ornate compass, rail, and luggage motifs.
- `train-back.webp`: vertical train-card back with a centered gilded railway crest, dark leather, embossed flourishes, and weathered brass.
